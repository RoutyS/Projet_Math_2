Projet Math :

Couleurs utilisées :

Enveloppe convexe (Jarvis March) : Bleu
Enveloppe convexe (Graham Scan) : Vert
Triangulation Incrémentale : Cyan
Triangulation Delaunay : Jaune
Diagramme de Voronoï : Rouge

Raccourcis clavier :

Clic gauche souris : Ajouter un point
Touche Espace : Calculer l'enveloppe convexe avec Jarvis March
Touche G : Calculer l'enveloppe convexe avec Graham Scan
Touche T : Calculer la triangulation incrémentale
Touche D : Calculer la triangulation de Delaunay
Touche A : Ajouter un point aléatoire avec Delaunay
Touche R : Supprimer un point aléatoire avec Delaunay
Touche V : Générer le diagramme de Voronoï

Pour chaque opération, le temps d'exécution est affiché dans la console grâce à MeasurExecutionTime
